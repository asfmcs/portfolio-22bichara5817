HTML Blog #1

The first thing that I learned in HTML was the basic script. First, we had to write !DOCTYPE html between a pair of these < >. This is so the computer knows its reading HTML and not any other language. Inside those, the computer needs to know what text is html, to we type html between a pair of these < >. Every command in html has to be typed between a pair of these < >. The computer then recoignizes this and decides what to do. After every command, you have to type the closing statement, which is just the repeat of the command but with a backslash. For example, when opening a paragraph, you type p and when closing a paragraph, you type /p.

CSS Blog #1

The first thing I learned in CSS is how to change color. To change color, you simply open a CSS style sheet in HTML and choose what you want to color (or what background). Type the HTML command without the < > inside the style sheet and type a curly bracket { }. Inside those brackets, type the command color followed by a colon : and then rgb. rgb sets the color of a pixel with red, green, and blue values. In the rgb section, type your r, g and b values from 0-255 to set the intensisy of the color in the pixel. This makes for a pretty versatile set of colors.
